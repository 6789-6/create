# Romance Orbit · 极光玫瑰宇宙

一个独立的浪漫主题 3D 交互作品。它不是诗云项目，而是一个可以旋转、缩放、点击光点进入细节层的浪漫星体原型。

## 项目目标

- 全屏沉浸式黑紫宇宙背景
- 可拖动旋转的 3D 浪漫星体
- 心形玫瑰星尘、极光轨道、月光轨道
- 点击光点切换浪漫记忆节点
- 所有信息以悬浮卡片呈现

## 技术栈

- React
- TypeScript
- Vite
- Three.js
- React Three Fiber
- Drei OrbitControls

## 本地运行

```bash
npm install
npm run dev
```

## 构建

```bash
npm run build
```
